Dataset
1. CIFAR-10
-> CIFAR-10 dataset은 32x32픽셀의 60000개 컬러이미지가 포함되어있으며, 각 이미지는 10개의 클래스로 라벨링
-> MNIST와 같이 머신러닝 연구에 가장 널리 사용되는 dataset중 하나

# 0 비행기
# 1 자동차
# 2 새
# 3 고양이
# 4 사슴
# 5 개
# 6 개구리
# 7 말
# 8 배
# 9 트럭

2. Fashion_mnist
Fashion-MNIST는 기존의 MNIST 데이터셋(10개 카테고리의 손으로 쓴 숫자)을 대신해 사용 가능

# 0 티셔츠/탑
# 1 바지
# 2 풀오버(스웨터의 일종)
# 3 드레스
# 4 코트
# 5 샌들
# 6 셔츠
# 7 스니커즈
# 8 가방
# 9 앵클 부츠
